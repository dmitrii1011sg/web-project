### Install

1. Clone repository
   ```sh
   git clone https://github.com/dmitrii1011sg/web-project.git
   ```
2. Install requirements
   ```sh
   pip install -r requirements.txt
   ```
3. Create .env 
   ```sh
   TOKEN="your token"
   ```
4. Create /db
